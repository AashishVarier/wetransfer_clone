Contains workflow without using GHCP agents.

🧱 Step 1 — Create Folder Structure Manually
    mkdir wetransfer-clone
    cd wetransfer-clone

    mkdir backend frontend
    touch README.md INSTRUCTION.md KNOWLEDGE.md

⚙️ Step 2 — Backend Boilerplate (Python + Serverless)
    1️⃣ Install Serverless Framework
        npm install -g serverless
    2️⃣ Initialize Python Service
        cd backend
        serverless create --template aws-python3 --name wetransfer-backend

    This creates:

    handler.py
    serverless.yml

    Now edit serverless.yml.

🎨 Step 3 — Frontend Boilerplate (React + Vite + Tailwind)
    1️⃣ Create React App (Vite)
        cd ../frontend
        npm create vite@latest

        Choose:
            React
            JavaScript

        Then:
        npm install

    2️⃣ Install Tailwind
        npm install -D tailwindcss postcss autoprefixer
        npx tailwindcss init -p

        Configure tailwind.config.js:

            export default {
            content: ["./index.html", "./src/**/*.{js,jsx}"],
            theme: {
                extend: {},
            },
            plugins: [],
            }

        In src/index.css:

            @tailwind base;
            @tailwind components;
            @tailwind utilities;

        Import it in main.jsx.
